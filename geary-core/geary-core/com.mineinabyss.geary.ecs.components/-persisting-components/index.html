PersistingComponents

data class PersistingComponents(components: MutableSet<GearyComponent>, hashed: Int) : MutableSet<GearyComponent>
Content copied to clipboard

A component that holds a list of components to be persisted.

It's not persisted itself, as we know which components are persistent when deserializing an entity by looking at the ones that were serialized to it.
