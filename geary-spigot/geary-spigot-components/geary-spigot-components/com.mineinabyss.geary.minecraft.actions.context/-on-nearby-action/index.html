Executes actions on nearby entities.

Parameters

radius

Will run on entities in a cube box extending radius blocks into each direction.

max

The maximum number of entities to select. Will sort by distance.

at

The location from which to start the search.

onSelf

Whether to include self within the found entities.

run

The actions to run on all entities found.
